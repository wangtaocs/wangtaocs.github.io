      {
      *node = *(node->next);
      }

三、PS

​ 我尝试过用java的方法，用C实现，但是我发现如果node是倒数第二个节点时，就会堆栈溢出，这肯定是node.next = node.next.next;这句代码出了问题。我觉得可能是官方给的数据最后一个节点的next没有置为NULL，导致指针在最后一个节点时，找不到他的下一个节点；而当我们自己写代码时，我认为始终令最后的一个节点的next置为NULL，这样才不会总是报错，这是一个良好的习惯。
