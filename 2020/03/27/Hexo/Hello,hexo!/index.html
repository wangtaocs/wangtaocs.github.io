Hexo

1.安装

  1. 安装Hexo脚手架：npm install -g hexo-cli
  2. 在任意目录下初始化：hexo init Blog

2.插件

2.1.hexagon

  • hexo的包管理工具

  • 安装：npm install -g hexagon-cli

  • 查看初始化后的hexo的所有插件和主题：hexa -l

  • ee

2.2.hexo-auto-category

  • 自动解析文章目录，方便博文归类整理
  • 安装：npm install hexo-auto-category --save

2.3.hexo-wordcount

  • 自动统计文章字数
